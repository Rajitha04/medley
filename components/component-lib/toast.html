Toast Component

Toast component is like an alert box that is only shown for a couple of seconds when something happens

Types of Toasts!

There are different types of toasts based on the result like error toast, success toast. The below toasts are created using display and position property and contains hover effects

Success

Your changes are saved successfully

Error

Error has occured while saving changes.

Info

New settings available on your account.

Warning

Username you have entered is invalid.
